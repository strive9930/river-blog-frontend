### 测试用例 5：权限功能验证

**步骤：**
1. 登录后访问任意页面
2. 检查权限指令是否生效

**测试代码：**
```vue
<template>
  <div>
    <!-- 应该显示（如果有权限） -->
    <button v-permission="'article:create'">发布文章</button>
    
    <!-- 应该显示（如果是管理员） -->
    <div v-role="'Admin'">管理员内容</div>
  </div>
</template>
```

**验证方法：**
```typescript
import { usePermission } from '@/composables/usePermission';

const { hasPermission, hasRole, isAdmin } = usePermission();

console.log('有创建文章权限:', hasPermission('article:create'));
console.log('是管理员:', isAdmin.value);
console.log('有 Admin 角色:', hasRole('Admin'));
```

## 🔍 自动化测试脚本

### 浏览器控制台脚本

```javascript
// 自动测试登录流程
(async function testLogin() {
  console.log('🧪 开始自动化测试...\n');
  
  // 1. 检查初始状态
  console.log('1️⃣ 检查初始状态');
  console.log('   Token:', !!localStorage.getItem('auth_token'));
  console.log('   用户信息:', !!localStorage.getItem('user_info'));
  
  // 2. 等待登录完成（需要手动登录）
  console.log('\n2️⃣ 请手动执行登录操作...');
  console.log('   登录完成后，继续观察以下输出\n');
  
  // 3. 定期检查状态（每 2 秒检查一次）
  const checkInterval = setInterval(() => {
    const token = localStorage.getItem('auth_token');
    const userInfo = localStorage.getItem('user_info');
    
    if (token && userInfo) {
      clearInterval(checkInterval);
      
      console.log('✅ 登录成功！\n');
      console.log('3️⃣ 验证数据完整性');
      
      try {
        const user = JSON.parse(userInfo);
        console.log('   - Token: ✅');
        console.log('   - 用户ID:', user.id ? '✅' : '❌');
        console.log('   - 邮箱:', user.email ? '✅' : '❌');
        console.log('   - 角色:', user.roles?.length > 0 ? '✅' : '❌');
        console.log('   - 权限:', user.permissions?.length > 0 ? '✅' : '❌');
        console.log('   - 菜单:', user.menus?.length > 0 ? '✅' : '❌');
        console.log('   - 管理员:', user.isAdmin !== undefined ? '✅' : '❌');
        
        console.log('\n4️⃣ 性能检查');
        console.log('   - 请求次数: 应该只有 1 次 /auth/login');
        console.log('   - 请在 Network 标签中验证\n');
        
        console.log('🎉 测试完成！');
      } catch (e) {
        console.error('❌ 解析用户信息失败:', e);
      }
    }
  }, 2000);
  
  // 30 秒后停止检查
  setTimeout(() => {
    clearInterval(checkInterval);
    console.log('⏰ 超时，未检测到登录状态');
  }, 30000);
})();
```

**使用方法：**
1. 打开登录页面
2. 在控制台粘贴并执行上述脚本
3. 执行登录操作
4. 观察测试结果
